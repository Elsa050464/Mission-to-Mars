# Mission-to-Mars
## Overview
The Mission of this project is to learn to scrape information pertaining to Mars using various applications to name:
1.	Python
2.	Jupyter Notebook 
3.	Pandas
4.	BeautifulSoup
5.	Splinter
6.	Mongo
7.	Flask
8.	HTML and 
9.	Bootstrap 
<br>
## Through the process, I learn how to scrape the following: 
**•	Latest News**
**•	Featured Image**
**•	Facts about the planet**
**•	Mars Hemispheres**
